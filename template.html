Инструкция по созданию новой страницы

ШАГ 1: Скопируйте и переименуйте этот файл

Сделайте копию этого файла (template.html) и дайте ей новое имя. Название должно быть на латинице, без пробелов, и заканчиваться на .html (например: novosti.html или obzor-igry.html).

ШАГ 2: Измените заголовок во вкладке браузера

Откройте новый файл в текстовом редакторе, поднимитесь в самый верх (в секцию <head>) и найдите строку <title>Название страницы - WEB ИГРЫ</title>. Замените "Название страницы" на нужное вам.

ШАГ 3: Замените этот блок на свой контент

Теперь удалите весь этот блок с инструкцией (он заключен в теги <div class="instruction-box">...</div>) и вместо него напишите свой текст, используя "Полезные теги" ниже.

Полезные теги для вашего текста:

  • <h1>Главный заголовок</h1> — Самый главный заголовок, используйте его один раз на странице.
  • <h2>Подзаголовок</h2> — Заголовок для разделения темы на части.
  • <p>Ваш текст...</p> — Обычный абзац текста.
  • <a href="адрес">Текст ссылки</a> — Ссылка. Например, <a href="page1.html">На главную</a>.
